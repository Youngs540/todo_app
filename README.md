# todo_app

Simple command-line todo app for tracking tasks using a JSON file.

Features
- Minimal, single-file Python script (`todo.py`).
- Stores tasks in `todo.json` so data persists between runs.

Requirements
- Python 3.8 or newer

Quick start
1. Ensure Python is installed.
2. Run the app:

```
python3 todo.py
```

Notes
- `todo.json` contains your saved tasks.
- Inspect or edit `todo.py` for available commands and options.
